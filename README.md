# Video Compressor – Fast Cross-Platform File Compression App 🚀

A professional, high-performance file compression utility built with Flutter, designed to seamlessly reduce the size of your videos, images, and PDF documents without compromising on quality.

## 📱 Project Description
Video Compressor is a modern, cross-platform application that empowers users to optimize their media and documents for storage and sharing. Whether you're dealing with bulky 4K videos, high-resolution photos, or large PDF reports, this app provides a lightweight and intuitive interface to get the job done quickly.

## ✨ Key Features
- **📹 Video Compression**: Efficiently reduce video file sizes while maintaining visual clarity.
- **🖼️ Image Optimization**: Compress high-res photos for faster web loading and storage savings.
- **📄 PDF Compression**: Shrink document sizes for easier email attachments and sharing.
- **🌍 Cross-Platform Support**: Unified experience across Android, iOS, Windows, Linux, and Web.
- **⚡ Fast Processing**: Optimized algorithms for hardware-accelerated compression.
- **💎 Lightweight UI**: Sleek, modern, and responsive interface designed for productivity.

## 🛠️ Tech Stack
- **Framework**: [Flutter](https://flutter.dev)
- **Language**: [Dart](https://dart.dev)
- **Storage**: Hive (Local persistence)
- **Integrations**: Platform-specific compression engines (FFmpeg/MethodChannels)

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed on your machine.
- Git installed.

### Installation Steps
1. **Clone the repository**:
   ```bash
   git clone https://github.com/appakuttyv/video_compressor.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd video_compressor
   ```
3. **Install dependencies**:
   ```bash
   flutter pub get
   ```
4. **Run the application**:
   ```bash
   flutter run
   ```
